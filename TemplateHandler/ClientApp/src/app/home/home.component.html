Handbook

Handbook will show you the utilisation of this templatehandler service.

User

You will be able to interact with the service under the user menu

  • Personal settings

    Under this option you will be able to change your personal user data, like name, email or password.

  •  Handle templates

    Under this option you will be able to maintain the templates which are visible to you, you can remove, update, upload template types.

  •  Executions

    Under this option you will be able to make complete files from template and data file, you can upload the data file and choose the template you want.
